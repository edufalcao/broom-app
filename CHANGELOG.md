# Changelog

All notable changes to Broom will be documented in this file.

## [Unreleased]

### Added
- System Cleaner: scan and remove caches, logs, temp files, browser caches, Xcode data, developer caches, .DS_Store files, mail attachments
- App Leftover Cleanup: detect orphaned files from uninstalled apps with confidence scoring
- App Uninstaller: full uninstall of apps with all associated Library files
- Drag-and-drop .app uninstall support
- Settings: launch at login, cleaning preferences, safe list, about
- Full Disk Access detection with permission banner
- Keyboard shortcut: Cmd+Shift+S to start scan
- Notifications after scan and clean operations
- Move to Trash by default (recoverable)
- Running app detection before cache cleaning
