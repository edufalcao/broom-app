## Broom v1.2.1 — SwiftUI Quality Pass

**Broom 1.2.1** improves concurrency safety, accessibility, and macOS integration with a Settings scene, type-safe routing, NavigationStack transitions, and better VoiceOver support.

---

### Settings & Navigation
- Settings now opens as a native macOS Settings window (Cmd+,) instead of a modal sheet
- Toolbar gear button uses SettingsLink for standard macOS behavior
- Category drill-in uses NavigationStack for proper push/pop transitions with scroll preservation

### Accessibility
- Category rows are proper Buttons for full VoiceOver and keyboard navigation
- Clean complete screen shows correct action ("moved to Trash" vs "permanently deleted")
- Uninstall confirmation sheet adapts height to content instead of fixed 460pt

### Performance
- Cached RelativeDateTimeFormatter in list rows — no longer allocated per row evaluation
- Smooth fade animations on all state transitions (scan, clean, done)

### Architecture
- Replaced six NotificationCenter publishers with type-safe AppRouter observable
- Added @MainActor to UninstallerViewModel for concurrency safety
- Added LargeFileScanning protocol for dependency injection
- Modern dropDestination API replaces legacy onDrop handler in App Uninstaller

### Quality
- 65 tests across 20 suites (up from 57/19)
- New AppRouter test suite (5 tests)
- ScanViewModel movedToTrash coverage (2 tests)
- LargeFilesViewModel DI coverage (1 test)
