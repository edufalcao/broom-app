## Broom v1.1.0 — Uninstall & Orphan Hardening

**Broom 1.1.0** delivers more complete app uninstalls, dramatically fewer false-positive leftovers, stronger safety enforcement at the deletion boundary, and faster app loading across the board.

---

### App Uninstaller
- Uninstalls now remove launch agents, login items, privileged helpers, package receipts, application scripts, and ByHost preferences
- LaunchServices metadata is cleaned after uninstall: the app is unregistered and the system database is refreshed
- Uninstall preview groups artifacts by source (User Data, Preferences, Caches, Launch Items, Helpers, etc.) with section headers
- Name variant generation discovers artifacts stored under non-standard names (hyphenated, underscored, version-trimmed)
- "Back to apps list" button on the uninstall success screen

### App Leftover Cleanup
- Orphan detection rewritten with suppression-first architecture: 9 gates filter candidates before they reach results
- Only stale, high-confidence leftovers are shown; recently active, running, or ambiguous items are excluded
- Stale-age threshold (default 30 days) is configurable in Settings
- Spotlight and receipt signals suppress false positives instead of inflating confidence
- Select All checkbox at the top of scan results
- Low-confidence items shown in a separate dimmed section for review

### Protected Data
- Sensitive app data is never surfaced in generic cleanup scans
- Protected families: password managers, VPNs, browsers, AI tools, iCloud-synced data, and automation tools

### Safety
- Every deletion is validated through DeletePolicy before executing: path safety, symlink resolution, and protected-data checks
- Structured DeleteResult (success/blocked/failed) replaces raw success/failure, making blocked operations visible
- Running app detection uses precise matching (full bundle ID, path components) instead of broad substring tokens
- Receipts under /var/db/receipts are accessible during explicit uninstalls but blocked in generic scans

### Performance
- App list loads significantly faster using Spotlight metadata for bundle sizes
- Associated files load on demand when an app is selected, not upfront for all apps
- Apple and system apps are filtered from the uninstaller list

### Quality
- 162 tests across 27 suites (up from 72 across 21)
- New test suites: DeletePolicy, ProtectedDataPolicy, UninstallArtifactPlanner, MetadataCleanup
