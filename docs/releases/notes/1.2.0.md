## Broom v1.2.0 — Large File Finder, Docker & Homebrew Cleanup, Enhanced Detection

**Broom 1.2.0** adds a new Large File Finder, expands the cleaner with Docker and Homebrew categories, improves orphan detection using macOS receipt and Spotlight databases, and completes the polish pass with Dock badge, accessibility, and a modal Settings sheet.

---

### Large File Finder
- New sidebar tab to scan your home directory for large files (100 MB+)
- Sort results by size, name, or modified date
- Configurable minimum size threshold (100 MB, 250 MB, 500 MB, 1 GB)
- Reveal any file in Finder with one click
- Per-file selection with Move to Trash

### System Cleaner
- Docker cleanup: scan Docker VM disk images and configuration (shown only if Docker is installed)
- Homebrew cleanup: detect old formula versions and cached downloads (auto-detects ARM vs Intel)
- App Leftovers now appear as a regular category row with drill-in detail
- Confidence badges preserved on each leftover item

### Orphan Detection
- Receipt database: reads /var/db/receipts to identify apps installed via .pkg
- Spotlight metadata: queries macOS for previously indexed bundle IDs
- Improved confidence scoring using receipts, Spotlight, and Saved Application State

### App Polish
- Dock icon badge shows total junk size after scan, clears after cleaning
- VoiceOver accessibility labels on all key interactive elements
- Settings presented as a modal sheet (blocks main window until dismissed)
- Safe List shows helpful empty state instead of a blank screen

### Quality
- 57 tests across 19 suites covering all new features

### Install
Download **Broom-v1.2.0.dmg** below, open it, and drag Broom to your Applications folder.
