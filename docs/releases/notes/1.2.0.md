## Broom v1.2.0 — Cleaner Leftovers & Uninstall UX

**Broom 1.2.0** eliminates false-positive system leftovers, fixes Group Container matching for apps like Microsoft Teams, and improves the uninstall tab with a clearer entry point.

---

### App Uninstaller
- Sidebar label renamed from "Apps" to "Uninstall" for clarity
- Uninstall tab starts with an idle screen and a "Scan Apps" button instead of auto-scanning on tab switch

### App Leftover Cleanup
- All `com.apple.*` entries are broadly suppressed from orphan detection — system and framework data is never listed as leftovers
- Group Container entries with team ID prefixes (e.g., `UBF8T346G9.com.microsoft.teams`) are now correctly matched against installed apps and suppressed

### Quality
- 168 tests across 27 suites

### Install
Download **Broom-v1.2.0.dmg** below, open it, and drag Broom to your Applications folder.
