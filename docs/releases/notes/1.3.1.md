## Broom v1.3.1 — Framework-Bundled System App Fix

**Broom 1.3.1** fixes classification of system apps embedded inside framework bundles under `/System/Library/Frameworks/`, ensuring they are correctly excluded from the uninstaller list.

---

### Fix
- System apps under `/System/Library/Frameworks/` (e.g., Wish.app inside Tk.framework) are now correctly classified as system apps and excluded from the uninstaller list

### Quality
- 178 tests across 27 suites
