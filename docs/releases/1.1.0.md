# Broom 1.1.0

Released on 2026-03-15.

## Highlights since 1.0.0

### Cleaner
- Orphaned app leftovers are now detected and merged into the main scan results flow.
- Cleaning settings now control developer-cache scanning, .DS_Store scanning, temp-file age filtering, and delete behavior.
- Running apps are detected before cleaning so active-app caches can be skipped or cleaned intentionally.

### Uninstaller
- Dragged `.app` bundles now open an uninstall preview even when they were not already indexed.
- App sizes include associated files, so sort-by-size reflects the full uninstall footprint.
- Uninstall previews now support per-file selection plus Trash or permanent-delete choices.

### App polish
- The Apps view keeps visible separators between the sidebar, list, and detail panes even when the window is inactive.

### Quality
- Added broader automated coverage for scanner, orphan detection, settings, cleaning, inventory, and uninstall flows.
